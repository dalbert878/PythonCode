# Inspirational Quotes App

## Overview

The Inspirational Quotes App uses Python's Kivy framework to fetch and display random quotes from the ZenQuotes API. It integrates text-to-speech functionality with threading for a seamless user experience.

### Features

- **Text-to-Speech**: Uses `pyttsx3` to vocalize quotes.
- **Threading**: Background processing for speech without blocking the UI.
- **Dynamic Quotes**: Fetches quotes and authors in real-time.
- **User Interface**: Intuitive layout with buttons to repeat and fetch new quotes.
- **Image Display**: Includes a visual element with `header.jpg`.

### Usage

1. Install dependencies:
   ```bash
   pip install kivy requests pyttsx3
   ```

2. Run the app:
   ```bash
   python main.py
   ```
